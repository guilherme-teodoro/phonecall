# phonecall

Gerenciador pessoal de números aleatores de celular para testemunho


## Subindo projeto
``` 
$ yarn 
$ yarn start
``` 

Abra [http://localhost:3000](http://localhost:3000) no seu navegador

## Buildando

``` 
$ yarn build
```
